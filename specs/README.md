# specs/

This directory is where spec-kit stores your feature specifications.

When you're ready to start building, run:

```
/speckit.specify <your hackathon idea>
```

This will create `specs/001-<name>/` with a structured spec, plan, and task breakdown generated from your description.

See [docs/spec-kit/README.md](../docs/spec-kit/README.md) for the full workflow.
